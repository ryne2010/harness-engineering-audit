# Docs Gardening And Knowledge Lane

Owns source/wiki/schema layers, ingest/query/lint workflows, docs health checks, indexes, and logs.

## Checks

- Raw sources are distinct from generated/synthesized docs.
- Indexes and logs are maintained.
- Stale claims, contradictions, orphan pages, broken links, and missing cross-references are checked.
- Docs search or navigation is documented when corpus size requires it.
