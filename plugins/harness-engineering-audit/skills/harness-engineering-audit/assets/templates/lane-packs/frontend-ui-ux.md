# Frontend UI/UX Lane

Owns design intent, visual source-of-truth assets, UI constraints, host/screen matrix, interaction states, accessibility criteria, and visual QA evidence.

## Grounded Sources

- Design targets and descriptions
- Visual assets: PNG, SVG, screenshots, videos, mocks, or inspiration references
- Component/system constraints
- Interaction and accessibility requirements
- Visual QA acceptance thresholds

## Host / Screen Matrix

Record applicable hosts: web, desktop, mobile, tablet, watch, TV, embedded, browser extension, and repo-specific surfaces.

## Handoff

Designer, vision, and frontend implementation agents must cite the target assets and return screenshot or visual-check evidence.
