# Docs Index

This directory is the source-of-truth documentation surface for the harness engineering audit repo.

## Core Docs

- `INSTALL.md` - install, pinned install, manual install, and explicit update commands.
- `USAGE.md` - audit execution, generated outputs, update checks, and recommended workflow.
- `PUBLISHING.md` - release and publishing notes.
- `PLUGIN_DISTRIBUTION.md` - optional plugin wrapper distribution.
- `RELEASE_CHECKLIST.md` - release validation checklist.
- `OMX_WORKFLOW.md` - OMX workflow expectations.

## Reference Docs

- `HARNESS_ENGINEERING_SOURCE_SYNTHESIS.md` - synthesized concepts from harness engineering, Symphony, and comparison sources.

## Validation

Run the repo validation entry point from the repository root:

```bash
make validate
```
